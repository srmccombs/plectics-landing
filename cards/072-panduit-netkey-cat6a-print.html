NetKey® Cat 6A UTP Punchdown Keystone Jack Module

High-speed punchdown keystone jack module provides reliable 10 Gigabit Ethernet connections for demanding office, workspace, and data center applications. Category 6A performance - Exceeds ANSI/TIA-568-C.2 Category 6A and ISO 11801 Class EA requirements at swept frequencies up to 500 MHz. 10GBASE-T ready - Supports 10 Gigabit Ethernet applications. Punchdown termination - Standard 110-style punchdown for quick, reliable connections. T568A/T568B wiring - Supports both standard wiring schemes. PoE support - Supports IEEE 802.3af/802.3at and proposed 802.3bt type 3 and type 4 PoE applications. Keystone module design - Compatible with NetKey faceplates and patch panels. Polycarbonate body - Durable construction. Plated phosphor bronze contacts - Reliable connectivity. 10 color options - Easy network color-coding and organization. UL 1863 approved. RoHS Compliant - Environmentally responsible.
